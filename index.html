Definition

Application that offers resources to the app drivers (Uber, 99pop), designed to help drivers make better decisions when accepting or not a ride, informing the value per KM/hour, to help them have a better experience at work.

How to Download the StopClub APP

  1. Go to your app store
  2. Look for StopClub
  3. Download the StopClub APP
  4. Create an account

How to Install the Macro for the Earnings Calculation to Work

  1. Click on Benefits
  2. Click on Earnings Calculation
  3. Click install MacroDroid
  4. Return to the StopClub APP
  5. Click on install Macro (Open with MacroDroid option)
  6. Ok
  7. To allow
  8. Yes to Accessibility Permission
  9. Installed Services
  10. Activate MacroDroid
  11. MacroDroid UI interaction
  12. Activate and Allow
  13. Overlay
  14. Authorize

After all the steps, you will see two floating buttons on your screen, this means that the Macro is working.

As soon as your next ride rings, you will be able to see the note, the price per Hour and the price per KM, if you have accepted the ride and forgotten the destination, the value and the passenger's note, don't worry, it will be in the your notifications.

Floating buttons

  1. Balloon Icon

    2. It will open the Google Maps on your destination.

  2. Gear Icon

    3. Configuration of the details that will be displayed on your screen.

    • Display of floating texts:

      • Passenger note, value per KM/hm, you can configure the value per KM/hm like your preferences.

    • Notification type:

      • By text or voice.

    • Set message for the passenger:

      • You will leave a message for the passenger, as soon as the trip starts, just go to the App you are using and paste the message for the passenger.

    • Traffic light colors

      • Green → Go for it
      Yellow → Maybe
      Red → It's not good, have a look before accept

    • Automatic decline:

      • You need to leave it ON, then just choose the value you want to cancel automatically.
      Example: If you choose the minimum value of $9.00, all rides below this value will be automatically cancelled.

Finishing

After all these settings, just go online in the mobility APPs (Uber/99) and start enjoying the value traffic light and automatic decline, like the settings you made.

We remind you that you can change your settings at any time.

100% free application, with option Earnings Calculation, Auto-Refusal, Secret Cam, Location Sharing, Chat and Live Streaming options.
